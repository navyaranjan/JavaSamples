# Kafka Producer - 001

This is a hobby project to familiarize Kafka Producer API and Twitter4J streaming API.

## Components

* twitter4j
* Spring Boot
* Apache Kafka

## Usage

* Clone the repository
  ```
  git clone https://github.com/navyaranjan/JavaSamples.git
  cd JavaSamples/kafka-producer-001
  ```

* Run with maven
  ```
  mvn spring-boot:run
  ```
